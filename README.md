## 🐉 What is Shin Sekai I.A?

**Shin Sekai (新世界)** means **"New World"** in Japanese.

Shin Sekai I.A is an **AI-powered document analysis assistant**, designed to allow users to interact with and extract insights from any type of file through natural conversation.

With a strong visual identity — black and gold theme with the 新世界 dragon — the system is built as a premium, focused, and secure solution for document intelligence.

> ⚠️ **Important:** This repository is a **DEMO version**.  
> It currently showcases **only the interface (UI/UX)** of the system.  
> Core features such as AI processing, document analysis, and full backend functionality are **not included**.

---

## 📌 What does it do?

In the full product, users can upload a document and immediately **chat with the AI about its content**.

The system is designed to:

* Read the document  
* Understand its structure and meaning  
* Answer questions based on its content  

It behaves like a **specialist who has fully read the document and is ready to assist**.

> ⚠️ **Note:** In this demo, these features are **simulated or not functional**, as only the interface is available.

---

## 🔐 Access Control

Access is restricted to **authorized users only**.

The system includes:

* Login interface  
* Authentication system  
* Permission validation  

Unauthorized users receive:

> **"User does not have permission to access the system."**

> ⚠️ In this demo, authentication may be **mocked or partially implemented**.

## 📂 Supported File Types

In the full version, Shin Sekai I.A is designed to handle a **wide range of file formats**, including:

* 📄 PDF, Word, PowerPoint  
* 📊 Excel (tables, data, charts)  
* 💻 Source code files  
* 🗄️ SQLite databases  
* 🖼️ Images  
* 🎧 Audio files  
* 📦 ZIP, APK, binaries, certificates  

> ⚠️ These capabilities are part of the **complete product**, not fully available in this demo.

## 🧠 Technical Architecture

| Layer          | Technology         |
| -------------- | ------------------ |
| Frontend       | React + Vite       |
| Backend        | Python + Flask     |
| Database       | SQLite (local)     |
| AI             | GPT-4              |

> ⚠️ This demo primarily represents the **frontend layer**.
